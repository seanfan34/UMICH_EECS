Compiling

MSI

./mu msi.m

This will generate a C source file named msi.C

Step 2. Compile C code generated by Murphi

make msi 

Step 3. Run

./msi

MESI

./mu msi_opt.m

This will generate a C source file named msi_opt.C

Step 2. Compile C code generated by Murphi

make msi_opt

Step 3. Run

./msi_opt
